Image loader

Tool for loading and unloading geotif images into qgis.

Details of images to load are stored in the 'image_details' table.

Images with 'load' checked are loaded into QGIS when load images is clicked

Loading images will remove anything in the 'image_loader' group.

The runs table is a summary of the image details table for quickly selecting which of these images to load.

Generating image details from folder

"Image_details","Find from folder..." will add a row to the image details table for every .tif file in folder and subfolders.

Loading from csv/txt

The tool can load image details from a csv or txt file.

The file needs a header. The tool will ignore case and '_' charactors in field names .

A column named filepath is required. This should contain the filepath of the raster image. This can be absolute or relative to the csv/txt.

The following columns will be used if present,otherwise details will be generated from filePath:

  • runID:run
  • imageId
  • fileName:used as layer_name
  • imageType:this and run can be used to generate groups
  • groups

Saving image details

Image details can be saved as a csv, txt or sqlite database(.db / .sqlite) file.

Filepaths will be saved as a relative path (from the folder containing the csv).

Relative paths allow a folder containing the csv and images to be moved without invalidating the file paths.

Frames layer

spatial fromes can be loaded trough load layers,load spatial frames data.

After filling in the 'setup,set layers and fields' dialog:right clicking on the image details table then 'select from layer' selects rows matching the selected features of the layer.
